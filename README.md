# study-guides-for-Kosuke

中3 現代文のテスト対策ウェブアプリ。

## 📖 テスト範囲
- **「問いかける言葉」**（国谷裕子）— 評論・論説文
- **「夢十夜」第一夜・第六夜**（夏目漱石）— 小説

## 🚀 使い方
`index.html` をスマホやパソコンのブラウザで開くだけ。インストール不要・オフラインでも動きます。

## 📱 アプリの中身
- **ホーム** … テスト範囲の確認と、おすすめ勉強ルート（チェックで進捗が記録される）
- **問いかけ** … 作者情報／6つの意味段落マップ／重要語句・概念／内容理解の一問一答／主題まとめ
- **夢十夜** … 夏目漱石の紹介／第一夜・第六夜のあらすじと要点／一問一答
- **カード** … 暗記フラッシュカード（タップで裏返し、覚えた／まだ で仕分け）
- **テスト** … 選択・穴埋めミックスの力だめしテスト（自動採点＋解説）

進捗は端末に自動保存されます（localStorage）。

## ⚠️ メモ
配布ワークの手書き部分（語句の意味など）は一部読み取りにくかったため、
本文・キーワードから確実な内容を中心にまとめています。
細かい語句はお手元のワークとあわせて確認してください。
